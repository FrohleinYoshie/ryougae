## このサービスの思い
私は生徒会会計に所属をしています。
業務内容としてサークル活動や生徒会企画にてかかった費用の支給を行うことです。
支給に際してATMからお金を下ろす必要があるが、その時にお札から小銭に金額を崩さなければならない時がある。
しかし、どのくらい金額を崩す必要があるのか計算しなければならない面倒がある。
このサービスはそのような面倒を解消することができるサービスです。
このサービスは会計業務の課題を改善するために開発しました。
